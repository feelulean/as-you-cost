[영업] 견적정보 상세정보 등록
견적 접수일/ 견적 제출일 : 일자 선택 가능함. 이후 부서별 업무 일정 등록 시, 입력된 견적 제출일이 업무 기한의 기준이 됨. 
당사 가격 경쟁력 : 상/중/하 중에서 선택

생산 공장 구분 : 신규/ 기존 중 선택. 기존 선택시 기존 공장 선택 가능. 선택한 기존 공장 기준으로 제조경비 및 판매관리비 계산 비율 초기값 IF 함
생산 법인 구분 : 신규/ 기존 중 선택. 기존 선택시 기존 법인 선택 가능.
수주 가능성 : 상/중/하 중에서 선택 가능
수주 가능성 사유 : Text 입력 가능
경쟁업체 : 경쟁업체에 대한 설명 입력 가능
개발 가능성 : 신규 제품 개발 또는 프로젝트 수주 가능성 등 입력
찾기 : 첨부할 파일 선택하여 첨부 가능
임시 저장 : 입력 데이터의 저장 
구성 요청 : 견적프로젝트의 개별 정보(상세 정보, 원가코드/판가, 수량/할인율, 담당자) 입력 완료 이후 원가팀에 알림

[조회] 견적정보 등록 - 일정
1
견적원가 > 견적정보 등록 > 일정(조회)
해당 프로젝트에 협의된 일정 및 업무 처리 현황을 조회할 수 있음.
견적BOM 작성 : 설계가 사양제품별 BOM을 모두 등록하고 등록 완료를 수행하면 업무 종료 표시됨
신규자재 설계원가 작성 : 설계원가 담당자가 신규 부품의 단가 입력 후 등록 완료 처리하면 업무 종료 표시됨
신규자재 구매추정가 작성 : 구매 부문 담당자가 신규 부품의 구매추정금액을 임력하고 등록 완료 처리하면 업무 종료 표시됨.
재료비 단가확정 : 원가관리 담당자가 설계원가 및 구매 담당자와 신규 부품의 단가 확인 후 등록 완료 처리하면 업무 종료 표시됨
인원/투자비 등록 : 생산기술 담당자가 신규 설비 투자비, 신규 생산라인에 대한 생산 직접 인원계획, 견적 제품의 생산라인 지정 및 예상 C/T 값을 입력하고 등록 완료 처리하면 업무 종료 표시됨
견적원가 산출 : 원가관리에서 견적원가 산출하면 업무 종료 표시됨

부품단가 등록 : 신규 부품 단가를 입력하는 화면으로 이동(견적용 BOM이 등록된 이후 가능)

비율 값은 프로젝트 정보에서 등록한 공장/법인 기준으로 비율 정보가 기준정보에 등록된 경우, 해당 정보를 초기값으로 보여줌.

제조경비 계산결과를 표시함. 원가 항목별 산출 금액을 확인 가능함
금형비: 제품 단위당 금형비 단가를 입력함
외주가공비 : 제품 단위당 금액을 입력 or 판가(예상 견적가) 기준 비율을 입력 가능함. 비율을 입력하면 판가 대비 비율값으로 외주가공비를 계산함. 
계산 완료 : 제조경비 산출 결과를 확인 후 ‘계산완료’를 클릭하여 제조경비 계산 단계를 완료함

‘조회’아이콘을 클릭하여 WACC산출을 위한 기본정보를 불러온 후 ‘계산’ 아이콘을 클릭하여 WACC 산출
WACC 산출을 위한 기본정보 입력

t : 해당 국가의 세율 국가코드 선택입력으로 생성

S : ‘자금조달 계획’에입력한 자기자본금 합계 자동생성

E (Rm) : 시장포트폴리오의 기대수익률 자동생성
