[영업] 현상정보 프로젝트 등록
신규 프로젝트의 상세 내역을 등록하기 위하여 기존에 생성한 프로젝트를 조회 가능함. 
프로젝트의 처음 상세 정보 입력을 위하여 필수 업무임. 찾기 버튼 누르면 2번 화면 팝업 됨. 
견적 프로젝트 중 수주 확정된 프로젝트 내역을 조회할 수 있음. 조회되는 프로젝트는 수주 확정 후 아직 현상원가 산출하지 않은 프로젝트임. 이미 현상원가를 산출한 프로젝트는 초기 화면에서 조회할 수 있음. 
프로젝트 코드 : 선택하면 해당프로젝트 생성시 입력한 기본 정보가 화면에 표시됨. 상세 정보 입력 가능함.

[영업] 현상정보 상세정보 등록
차수명: 현상원가 산출 차수에 대한 설명을 입력할 수 있음.

생산 법인 구분 : 신규/ 기존 중 선택. 기존 선택시 기존 법인 선택 가능.
비고 : 현상원가 산출 프로젝트 및 원가산출 등에 대한 상세 설명 입력 가능함.
첨부파일 : 관련 파일 첨부 가능함
찾기 : 첨부할 파일 선택하여 첨부 가능
임시 저장 : 입력 데이터를 저장함
협의체 구성완료 : 현상원가 산출 업무를 시작하도록 함. 구성 완료 이후에는 원가 산출 제품(원가코드)를 변경할 수 없음. 구성완료 이후 원가 산출 제품 추가 등 변경이 필요한 경우에는 현상원가 차수를 변경하여 다음 차수에서 제품 추가 또는 삭제 등이 가능함
닫기 : 상세 정보 입력 화면 닫음

[영업] 현상원가 정보 수량/할인율 등록
연도 표시 : ‘원가코드/판가’ 탭에서 입력한 SOP / EOP 기준으로 표시됨
순번 : 현상원가 산출 대상 제품 순서로 표시
원가코드 : 향후 자동 생성된 원가코드 표시됨. 초기 입력 단계에서는 생성 전이므로 표시 안됨. 
매출 수량 : 연도별 RFQ 수량 입력. 초기에는 이전 현상 차수(Ex, 현상 3차 업무시 현상 2차 단계, 현상 1차 업무시 견적단계를 가리킴) 때의 값을 기초값으로 가져옴. 
할인율 : 연도별 판가 할인율 입력. % 값 기준으로 입력함(%는 입력 불가 : 엑셀의 할인율 값을 값복사 하는 경우, 복사한 데이터에 %가 들어있는 경우 데이터 저장시 에러 발생함). 초기에는 견적(또는 이전 현상 차수) 때의 값을 기초값으로 가져옴.
입력 데이터 : [원가코드/판가] 탭에서 [산출 대상] 열에서 선택한 항목에 대하여만 수량 및 할인율을 입력 가능함. 
연도 : [원가코드/판가] 탭에서 입력한 SOP / EOP 기준으로 표시됨

1

[태스크 명]
현상 BOM 작성 : 설계가 사양제품별 BOM을 모두 등록(PDM e-BOMIF)하고 등록 완료를 수행하면 업무 종료 표시됨
신규자재 설계원가 작성 : 설계원가 담당자가 신규 부품의 단가 입력 후 등록 완료 처리하면 업무 종료 표시됨
신규자재 구매추정가 작성 : 구매 부문 담당자가 신규 부품의 구매추정금액을 임력하고 등록 완료 처리하면 업무 종료 표시됨.
재료비 단가확정 : 원가관리 담당자가 설계원가 및 구매 담당자와 신규 부품의 단가 확인 후 등록 완료 처리하면 업무 종료 표시됨
인원/투자비 등록 : 생산기술 담당자가 신규 설비 투자비, 신규 생산라인에 대한 생산 직접 인원계획, 원가산출 제품의 생산라인 지정 및 예상 C/T 값을 입력하고 등록 완료 처리하면 업무 종료 표시됨
현상원가 산출 : 원가관리에서 현상원가 손익계산서 산출하면 업무 종료 표시됨

2
프로젝트 검색 조건 : 프로젝트 코드및 프로젝트 등록시 입력한 사업부, 고객사, 차종 등으로 프로젝트 내역 검색 가능함.
